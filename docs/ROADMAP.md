# 路线图（MVP -> 上线）

## 阶段 0 - 初始化
- 目录结构（扩展 + 文档）
- 构建工具（Vite 或最小依赖）
- Chrome 本地加载（unpacked）

验收：
- 扩展可加载，无报错
- 弹窗可打开

## 阶段 1 - 核心扫描
- Content Script 读取：title/description/OG/分享/canonical/robots/H1
- 共享解析与像素宽度计算
- 弹窗与内容脚本消息通信

验收：
- 弹窗能展示原始字段
- 正常网页可扫描

## 阶段 2 - 弹窗 UI（快速预览）
- 头部 + 状态 + 操作
- SERP 预览（桌面/移动切换）
- OG 缩略预览
- Top 3 问题列表

验收：
- UI 符合 docs/PRD.md 弹窗布局
- 打开弹窗自动扫描

## 阶段 3 - 完整预览页
- 扩展页面承载完整预览
- 左预览（SERP + OG）/右诊断
- 元信息详情可折叠

验收：
- “完整预览”可打开
- 诊断按严重度分组

## 阶段 4 - 规则引擎
- 实现 Free + Pro 规则（docs/RULES.md）
- 统一问题结构（id/level/title/detail/suggestion）

验收：
- 规则输出一致
- Free/Pro 边界生效

## 阶段 5 - 导出（Pro）
- 导出 PNG（预览区域）
- 复制文本摘要

验收：
- PNG 输出符合导出规范
- 摘要复制可用

## 阶段 5.5 - 付费与授权（MVP）
- Pro 功能开关（导出/高级诊断）
- 授权验证 API（Vercel Functions）
- 授权激活入口与升级引导

验收：
- 未激活时高级诊断/导出不可用
- 输入 License 可激活并记录到期

## 阶段 6 - 打磨与发布
- 错误态/加载态
- 商店素材

验收：
- 正常使用无控制台报错
- 可提交 Chrome Web Store
