# 支付流程（MVP，手动续费）

## 目标
- 仅人民币（支付宝）
- 手动续费（无自动订阅）
- 支付成功后开通 Pro

## 建议流程
1) 用户选择 Pro（月付/年付）
2) 跳转支付页（支付宝）
3) 支付成功后生成 License Key
4) 用户在插件内输入 License Key 激活
5) 验证通过后开通 Pro

## 最小数据
- order_id
- plan（月付/年付）
- 用户标识（邮箱或 license key）
- status（paid/unpaid）

## 授权模型（简化）
- 支付后生成 license key 并入库
- 插件调用授权验证接口开通

## 授权与有效期（MVP）
- 付费后开通期限：
  - 月付：30 天
  - 年付：365 天
- 续费：到期后需手动续费，续费即延长有效期
- 过期：到期后自动回退为 Free 功能
- MVP 触发方式（建议）：支付成功后通过手动发放 license key 或手动标记开通

## 备注
- 文案明确：到期需手动续费
- 不做自动续费开关
- 授权验证接口：`/api/license/verify`
- 手动发放时更新 `LICENSE_KEYS` 环境变量
