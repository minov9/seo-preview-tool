# 插件文案模板（极简中文）

## 1. 文案基调
- 专业、克制、短句
- 避免感叹号与情绪化
- 仅说事实与操作

## 2. 弹窗（Popup）
### 顶部区
- 标题：SEO 预览
- 子标题：{domain}
- 状态：
  - 正在扫描…
  - 已更新
  - 发现 {count} 个问题

### 操作按钮
- 刷新
- 完整预览

### 预览区
- SERP 预览
- OG 预览
- 设备切换：桌面 / 移动
- 标签：缺失 / 过长 / 过短

### 诊断区
- 问题
- 查看全部

### 异常提示
- 无法扫描此页面
- 未检测到元信息
- 图片加载失败

## 3. 完整预览页
### 顶部区
- 标题：SEO 预览
- 子标题：{url}
- 状态：发现 {count} 个问题

### 操作按钮
- 刷新
- 导出 PNG（Pro）
- 复制摘要（Pro）

### 预览区
- SERP 预览
- 分享卡片预览
- 设备切换：桌面 / 移动
- 像素提示：{width}px

### 诊断区
- 问题总览
- 严重 / 警告 / 提示
- 查看建议

### 详情区
- 元信息详情
- 复制

### Pro 引导（灰态按钮旁）
- 升级 Pro 解锁导出与高级诊断

## 4. 通用占位
- 无数据：暂无数据
- 加载中：加载中…
- 空列表：暂无问题
- 无预览图：无预览图
